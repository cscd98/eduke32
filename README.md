# EDuke32 (libretro core)

This repository contains the **libretro port of EDuke32**, based on the 1996 first‑person shooter **Duke Nukem 3D**.

The standalone upstream project is maintained at: https://voidpoint.io/terminx/eduke32

This project incorporates BUILD engine technology originally created by Ken Silverman. Further information regarding this can be found at:
[source/build/buildlic.txt](source/build/buildlic.txt)

---

## Building the libretro Core

```bash
cd platform/Libretro
make -f Makefile.libretro
```
